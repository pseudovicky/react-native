## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/pseudovicky/react-native.git

# Navigate into the project folder
cd react-native

# Install dependencies
npm install

# Run the app
npx react-native run-android   # or run-ios if you're on macOS
